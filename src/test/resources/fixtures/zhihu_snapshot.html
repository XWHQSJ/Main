如何看待 2024 年 Java 生态的发展？

Java 21 已正式发布，虚拟线程、模式匹配等特性逐步成熟。Spring Boot 3.x 全面拥抱 Jakarta EE。GraalVM 原生编译也在改善启动速度。大家怎么看 Java 在云原生和 AI 时代的前景？
虚拟线程是 Java 近十年最大的变化。Project Loom 让 Java 在高并发场景下不再需要响应式编程的复杂性，直接用同步代码就能达到异步的性能。Spring Boot 3.2 已经原生支持虚拟线程。
GraalVM 原生编译确实解决了 Java 冷启动慢的问题。Quarkus 和 Micronaut 在这方面走得更远。但说实话，对于大部分企业应用，JIT 编译后的 HotSpot 性能已经足够好了。
模式匹配让 Java 的表达能力大幅提升。sealed class + switch pattern matching 写出来的代码已经很接近函数式语言的感觉了。record 也让值对象的定义变得简洁。
